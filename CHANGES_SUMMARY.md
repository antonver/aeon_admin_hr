## Технические детали

### Алгоритм анализа результатов
```python
def analyze_interview_results(results_text: str) -> str:
    if not results_text:
        return "ожидает"
    
    lower_results = results_text.lower()
    if "не берем" in lower_results:
        return "не берем"
    else:
        return "берем"
```

### Новые эндпоинты
